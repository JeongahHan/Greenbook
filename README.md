# Greenbook

# I.프로젝트 정보
1. 소개
> 사용자들끼리 '책'을 사고 팔수 있는, 온라인 중고 책 거래 플랫폼입니다.

2. 제작기간
> 기간 : 2023.08.17 ~ 2023.08.31 (2주)

3. 주요기능
+ **중고거래 게시판 : 책 구매 or 판매** 
    + 관심상품 찜 기능
    + 구매요청 기능
    + 댓글 작성 기능
+ **독서후기 공유 게시판**
    + 자유로운 글쓰기 및 댓글 작성 기능
    + 댓글 內 좋아요 버튼 기능
+ **외부 서점의 베스트셀러 연동으로, 매일 무슨 책이 인기있는지 확인**
+ **통합검색 기능으로, 중고나라게시판 內 매물과 독서후기게시판 內 후기 동시 확인**
+ **유저에게 쪽지보내기 기능**
